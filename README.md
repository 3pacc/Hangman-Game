# Jeu du Pendu en Python avec Tkinter

Ce projet est une implémentation du célèbre jeu du Pendu en Python en utilisant le module Tkinter pour l'interface utilisateur.

## Présentation

Le jeu du Pendu est un jeu de devinette où un joueur doit deviner un mot en essayant de trouver les lettres une par une. Si le joueur échoue à deviner le mot après un certain nombre d'essais, une image du pendu est dessinée, représentant le nombre d'erreurs commises.

## Fonctionnalités

- Interface graphique conviviale grâce à Tkinter.
- Sélection aléatoire de mots à deviner.
- Affichage visuel du pendu.
- Possibilité de rejouer après avoir gagné ou perdu.

## Prérequis

Avant de jouer à ce jeu, assurez-vous d'avoir Python installé sur votre système. Vous pouvez le télécharger depuis [le site officiel de Python](https://www.python.org/downloads/).

## Comment jouer

1. Clonez ce dépôt sur votre machine.
2. Ouvrez le terminal et accédez au répertoire du jeu.
3. Exécutez le jeu en utilisant la commande suivante : `python hangman.py`.
4. Le jeu vous guidera tout au long du processus de jeu.

Amusez-vous à jouer au jeu du Pendu développé en Python avec Tkinter !
